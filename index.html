Glossary

Root Number / 根數 / 生命靈數

個人的核心數字，代表與生俱來的本質、天賦與主要性格特質，是解讀生命藍圖的起點。

Life / 生活

象徵個人能量在日常中的具體展現，同時也是生活運作的起點能量。它反映你如何運用天賦、心念與意圖去塑造現實，並為整體人生狀態奠定基調。生活的能量啟動點，決定了你向外展開的方向與節奏。

Subconscious / 潛意識

驅動外在行為與思維模式的深層內在力量，通常不自覺地影響個人的選擇、習慣與情感反應。

Inner World / 內心世界

個人內在的感受、思考、價值觀與精神活動的總和，是自我對話與情感體驗發生的內在空間。

Missing Number / 缺數

在個人生命數字圖（Numerology Chart）中完全未出現的數字。缺數所代表的能量與課題，往往是此生需要主動學習與發展的領域。當這份能量在潛意識或內心世界中偶爾浮現卻未在現實中充分展現時，可能引發內在的矛盾感與拉扯。

Core Combination / 核心數字組合

由生命藍圖中心三角數字構成的關鍵組合，主導了個人的核心性格、人生路徑與主要天賦。

Other Combinations / 其他數字組合

除核心組合外，圖表中其他數字間形成的互動與組合。這些組合代表了個人性格的不同側面、潛在能力以及在特定情境中可能展現的特質。
